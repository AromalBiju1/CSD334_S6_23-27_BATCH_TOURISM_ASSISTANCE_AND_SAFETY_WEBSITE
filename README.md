# 🧭 Tourist Assistance & Safety Website

[![Live Demo](https://img.shields.io/badge/🌐_Live_Demo-Visit_Site-brightgreen?style=for-the-badge)](https://csd-334-s6-23-27-batch-tourism-assi.vercel.app/)

A smart, safety-focused web platform designed to enhance the travel experience by combining tourist hotspot discovery with real-time safety-aware navigation and emergency support tools.

---

## 📌 Problem Statement

Tourists visiting unfamiliar locations often face:
- Limited access to reliable local information
- Unsafe or high-risk travel routes
- Lack of real-time emergency assistance

Most existing platforms focus only on navigation or attraction discovery, but **fail to integrate safety insights with travel planning**, leaving travelers vulnerable in unfamiliar environments.

---

## 🎯 Project Objective

To build a **user-friendly tourist assistance website** that helps travelers explore destinations **safely and confidently**.

### Key Objectives:
- Help users discover relevant tourist hotspots
- Reduce travel risk by identifying unsafe zones
- Suggest safer alternative routes
- Provide quick access to emergency assistance

---

## 🚀 Core Features

### 1️⃣ Tourist Hotspot Recommendation
- User enters a destination
- System returns a curated list of tourist attractions based on:
  - Popularity
  - Proximity
  - User preferences

### 2️⃣ Safety-Aware Navigation
- Interactive map-based navigation
- Route analysis using safety data
- Area classification:
  - 🟢 Green Zone – Safe
  - 🟠 Orange Zone – Moderate Risk
  - 🔴 Red Zone – High Risk
- Automatically suggests safer alternate routes when risks are detected

### 3️⃣ Emergency Support Tools
- One-tap access to local emergency contacts
- Location-aware help resources for critical situations

---

## 🗺️ Map & Location Services

- **Map Provider:** OpenStreetMap (OSM)
- **Map Rendering & Interaction:** Leaflet.js
- Displays:
  - Tourist hotspots
  - Risk zones (color-coded)
  - Safe and alternative routes
- Lightweight, open-source, and privacy-friendly mapping solution

---

## 🛠️ Technology Stack

### Frontend
- **Framework:** React.js
- **Styling:** Tailwind CSS
- **Map Library:** Leaflet.js
- **Map Data:** OpenStreetMap (OSM)

### Backend
- **Framework:** FastAPI (Python)
- **Database:** PostgreSQL

### APIs & Integrations
- Crime Data APIs (for real-time danger zone detection)
- Geolocation & routing APIs
- REST APIs for frontend–backend communication
